音声メモをInboxに保存してください。

内容: $ARGUMENTS

## 実行フロー
1. `brain/00_Inbox/作業中.md` を読む
   - 作業中の課題が記録されていれば → そのメモに紐づける
   - 記録がなければ → 「セッション外」として記録する
2. 入力テキストを整理する（箇条書きに構造化）
3. タグ・カテゴリを自動推定する
4. `brain/00_Inbox/` に保存する
   - フォーマット: `YYYY-MM-DD_voice-note_{連番}.md`
5. 関連する既存ノートがあればリンクを追加する
6. 確認を求めず即保存する。リレーションが違う場合はユーザーが後から修正する

## メモのテンプレート
```markdown
# Voice Note - {日付}

## 作業中の課題
{作業中.md から読んだ課題名 / セッション外}

## 内容
{構造化されたメモ}

## タグ
{自動推定されたタグ}

## 関連ノート
{既存ノートへのリンク}
```
