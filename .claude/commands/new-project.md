新しいプロジェクトを作成してください。

プロジェクト名: $ARGUMENTS

## 実行フロー
1. brain/20_Projects/{プロジェクト名}/ ディレクトリを作成する
2. 以下のテンプレートファイルを生成する:
   - `README.md` - プロジェクト概要、ゴール、スコープ
   - `decisions.md` - 意思決定ログ
   - `notes.md` - 作業メモ
3. GitHub Milestone を作成する
4. 初期タスクがあれば /decompose を実行する

## テンプレート (README.md)
```markdown
# {プロジェクト名}

## ゴール
（達成したい状態を記述）

## 背景・動機
（なぜこのプロジェクトをやるのか）

## スコープ
- IN:
- OUT:

## 状態
- [ ] 計画中
- [ ] 進行中
- [ ] 完了
```
