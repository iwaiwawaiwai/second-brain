# メルカリ大量出品自動化プロジェクト

## 概要
祖父母の遺品（約100点）をメルカリWebで効率的に出品するための半自動化。
料理・旅行趣味だった祖父母の遺品整理が背景。

## データ構造（7カラム、TSV形式）

| # | カラム | 内容 | 制約 |
|---|--------|------|------|
| 1 | ID | 4桁数字（フォルダ名と一致） | |
| 2 | タイトル | ID込みの完全版（例：`0132 木彫り霊芝置物 縁起物`） | 40文字以内 |
| 3 | カテゴリー | メルカリ正式名称 | |
| 4 | 商品の状態 | 定型文（メルカリ選択肢と完全一致） | |
| 5 | 販売価格 | 半角数字 | |
| 6 | 価格根拠 | 相場・根拠・参考元をコンパクトに | 例：`相場¥800-1,500（sold12件）→状態良好で¥1,200｜参考:メルカリ・ヤフオク` |
| 7 | 商品説明文 | エピソード＋スペック＋ハッシュタグ | 最大1000文字 |

タイトル列 = `{ID} {商品名}`（最大40文字、メルカリ制限に合わせる）

### 価格根拠カラムのフォーマット
```
相場¥{下限}-{上限}（sold{件数}件）→{判断理由}で¥{設定価格}｜参考:{情報源}
```
- 情報源例：メルカリ・ヤフオク・Amazon・楽天など
- sold件数が取れない場合は「sold数件」などで略記

### 商品の状態（定型文）
- 新品、未使用
- 未使用に近い
- 目立った傷や汚れなし
- やや傷や汚れあり
- 傷や汚れあり
- 全体的に状態が悪い

## ハッシュタグ優先順位
名称 > 用途 > ジャンル > 背景（遺品整理・断捨離系）

## ブックマークレット（現行版）

```javascript
javascript:(function(){try{
  var r=prompt("スプレッドシートの7列をコピーして貼り付けてください");
  if(!r)return;
  var c=r.split('\t');
  if(c.length<7)return;
  // c[0]=ID, c[1]=タイトル, c[2]=カテゴリー, c[3]=状態, c[4]=価格, c[5]=価格根拠, c[6]=説明文

  var v=(s,val,t)=>{
    var e=document.querySelector(s);
    if(e){
      var p=t==='textarea'?window.HTMLTextAreaElement.prototype:window.HTMLInputElement.prototype;
      var sd=Object.getOwnPropertyDescriptor(p,'value').set;
      sd.call(e,val);
      e.dispatchEvent(new Event('input',{bubbles:true}));
      e.dispatchEvent(new Event('change',{bubbles:true}));
    }
  };

  v('input[name="name"]', c[1]); // タイトル列にIDを含む完全版が入っている
  v('input[name="price"]', c[4]);
  v('textarea[name="description"]', c[6].replace(/\\n/g,'\n'), 'textarea');

  var cl=(txt)=>{
    var btns=document.querySelectorAll('button, div, li, span');
    for(var b of btns){ if(b.innerText===txt){b.click(); break;} }
  };
  cl(c[3]); // 商品の状態（c[5]の価格根拠はスプレッドシート記録用のため入力スキップ）

  alert("入力完了：タイトル・説明・価格・状態。カテゴリーは候補から選択してください。");
}catch(e){alert("Error:"+e.message);}})();
```

## 新規セッション開始テンプレート

新しいタブでClaudeに依頼するときはこれをコピーして送る：

```
メルカリ出品データ作成をお願いします。
仕様は brain/20_Projects/mercari-auto-listing.md を参照してください。

ID: 0001
サイズ: （不明なら省略OK）
素材: （不明なら省略OK）
型番: （なければ省略OK）
状態メモ: （気になる点があれば）
希望価格: （なければ相場調査して提案）

※写真を添付、または複数商品は --- で区切って続けて書く
```

## データ生成フロー

1. ユーザーが **ID＋知っている情報**（写真・素材・サイズ等、あるだけでOK）を提供
2. Claude がWebで相場調査しながら7列TSVを生成
3. 精度向上に必要な情報があれば、**その商品限定でピンポイント質問**（汎用チェックリストは出さない）
4. スプレッドシートに貼り付け → ブックマークレット実行
