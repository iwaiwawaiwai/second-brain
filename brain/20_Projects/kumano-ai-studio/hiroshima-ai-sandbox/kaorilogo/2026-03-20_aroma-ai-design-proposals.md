## 選定基準チェックリスト

デザイン選定時に以下の観点で各案を評価する。

| 観点 | 説明 |
|------|------|
| **転換率** | 検索 → 購入のCVRへの寄与 |
| **横展開性** | 他ブランドへのホワイトラベル適応しやすさ |
| **LLM親和性** | AI応答の自然な組み込みやすさ |
| **モバイル対応** | スマホ操作体験の質 |
| **ブランド差別化** | 競合ECとの差別化度 |

評価: ◎ 優秀 / ○ 良好 / △ 要検討 / × 課題あり

## 案01 ── 白衣の処方箋

**コンセプト:** 「香りの診断書」。医療・薬局の無機質な清潔感をラグジュアリーに昇華。

### ビジュアル方向
- **背景:** 純白 (#FFFFFF) + ライトグレー (#F5F5F0)
- **アクセント:** 薬品瓶のようなダークグリーン (#1A3A2A) と ゴールド (#C9A84C)
- **フォント:** モノスペース系見出し（Courier Prime Bold）+ サンセリフ本文
- **UI素材:** 細い実線ボーダー、グリッド構造、番号付きステップ

### インタラクション
1. 入力欄は「診断フォーム」風の一問一答
2. AIが分析中は「検査中...」のプログレスバー（医療機器風アニメーション）
3. 結果は「処方箋カード」として表示（香り名・効能・使用シーン）

### キーUI要素
```
┌─────────────────────────────────┐
│  Rx  アロマ診断  No.0032        │
├─────────────────────────────────┤
│  症状を記述してください          │
│  > 不安で眠れない夜              │
├─────────────────────────────────┤
│  診断結果 ──────────────────── │
│  ① Lavender    適合度 94%      │
│  ② Chamomile   適合度 87%      │
└─────────────────────────────────┘
```

## 案02 ── 深夜のラグジュアリーバー

**コンセプト:** 高級バーのバーテンダーが香りをブレンドするような、ダークで官能的なUI。

### ビジュアル方向

## 案03 ── 森の中の地図

**コンセプト:** 香りを「地形図」として空間的に探索する。チャットではなく「旅」のメタファー。

### ビジュアル方向
- **背景:** オフホワイト + 薄いサググリーン (#E8F0E4)
- **アクセント:** テラコッタ (#C4714A) とモスグリーン (#4A6741)
- **フォント:** 手書き風見出し（Caveat） + 読みやすいサンセリフ
- **UI素材:** 手描き風のボーダー、水彩テクスチャ、地図の等高線

### インタラクション
1. **香りマップ**: 画面の大部分が2Dマップ。香りがノード（点）として配置
2. キーワード入力 → AIがマップ上の「現在地」を判定し、周辺の香りをハイライト
3. ノードをタップで詳細カード展開

### キーUI要素
```
[森林系]──────[フローラル系]
    │                │
  Cedar           Rose ●← ここが近い
    │           Jasmine ●
[スパイス系]──────[シトラス系]
```

## 案04 ── 感情の輪（エモーション・ホイール）

**コンセプト:** ロバート・プルチックの感情の輪にインスパイアされた、感情から香りを逆引きするUI。

### ビジュアル方向
- **背景:** ライトグレー (#FAFAFA)
- **ビジュアル中心:** 8色のグラデーション円環（感情分類ごとに色分け）
- **フォント:** モダンサンセリフ（Inter / Noto Sans JP）
- **UI素材:** スムーズなSVGアニメーション

### インタラクション
1. 起点は「今の気分は？」という大きな感情ホイール
2. 感情（不安・リフレッシュ・ロマンティック…）をタップ
3. AIがさらに絞り込み質問（強度・シーン）
4. 最終的に香りカードをレコメンド

### キーUI要素
- フリーテキスト入力もあり（「なんか言葉にならない感じ」でも受け付ける）
- ホイール自体が検索結果を反映して色が変化

## 案05 ── 分子構造の科学館

**コンセプト:** 香りを「化学」として可視化する、理系クリエイター向けのデータビジュアル系UI。

### ビジュアル方向
- **背景:** #060A14（ほぼ黒）
- **アクセント:** エレクトリックブルー (#00C8FF) とライムグリーン (#7FFF00)
- **フォント:** IBM Plex Mono（全面モノスペース）
- **UI素材:** 粒子アニメーション（Three.js）、データグラフ

### インタラクション
1. テキスト入力 → LLMが成分キーワードを抽出し、3D分子グラフとして描画
2. 分子ノード（香料）をドラッグで組み合わせ → ブレンド試作
3. 相性スコアをリアルタイム計算して表示

### キーUI要素
- Linalool (72%) ●──● Linalyl Acetate (18%)
- ベクトル類似度の可視化バー

## 案07 ── チャットボット・コンシェルジュ

**コンセプト:** LINE/iMessageライクなチャットUIで「香りソムリエ」と会話しながら選ぶ。

- **アクセント:** ブランドカラー1色 + グリーン（送信）

### インタラクション

## 案09 ── ゲーミフィケーション・香りデッキ

**コンセプト:** トレーディングカードゲームのように香りを「集める」体験。コレクション欲を刺激。

### ビジュアル方向
- **背景:** ダークパープル (#1E0A3C) + ホログラフィックグラデーション
- **アクセント:** カード枠に光るエフェクト
- **フォント:** ファンタジー系ディスプレイ + 読みやすい本文

### インタラクション

3. 「デッキ（お気に入り）」に入れた香りの組み合わせでブレンドレシピ生成
4. 購入でリアルカードをプレゼント（オフライン連動）

- ホログラフィックカードのCSS（transform 3D + グラデーション）

## 案10 ── ミニマル・グリッドギャラリー

**コンセプト:** MoMA/ハイエンドファッションECのような、商品写真が主役の編集的UI。

### ビジュアル方向
- **背景:** 白 (#FFFFFF) + 余白大量
- **アクセント:** タイポグラフィのみで差をつける（色は使わない）
- **フォント:** Helvetica Neue / 游ゴシック — 極細〜太の対比
- **UI素材:** 整然とした12列グリッド、フルブリードの商品写真

### インタラクション
1. 上部に細い検索バー（目立たせない）
2. グリッドに商品が並んでいる（デフォルト状態）
3. 検索語を入力すると、グリッドがリアルタイムでソート・フィルタ
4. AIの提案は右端に薄いサイドバーとして表示（主役は商品）

### キーUI要素
- ホバーで香りのキーワードがオーバーレイ
- 「編集部おすすめ」のクレジット付き記事型レイアウト
- パーソナルカラー診断のようなクイズ導線
