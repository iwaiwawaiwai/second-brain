## 背景

**広島AIサンドボックス**（広島県のAI活用支援プログラム）に掲載されている企業課題に対して、AIエンジニアとして提案・開発を行う飛び込み営業型の案件。

- 対象企業: **株式会社KAORI LOGO**（広島市中区）
- 担当者: 森 かおり 代表
- サイト: https://kaorilogo.co.jp / https://lifeofaroma.com
- 事業: アロマ空間デザイン・天然精油商品の販売（Life of Aromaブランド）

---

## 企業の課題

> 香りは文字化が難しく、連想するキーワードも個人差が大きい。
> 現状は成分・好み・気分など複数のチェック項目で検索してもらっているが煩雑。
> 自然なキーワード入力だけで香りを探せるようにしたい。

### 現状の問題構造

```
対面カウンセリング: カウンセラーが質問 → 顧客が答える → 提案
    ↓ オンラインでは
サイト: カテゴリを見せる → 顧客が選ぶ → 商品ページへ
```

- 知識がある顧客はいいが、初心者には扱いにくいサイト構造
- 検索窓を置くだけでは「何を入力すればいいかわからない」問題が残る

---

## 提案方針

### 開発アプローチ: **C案（ハイブリッド型）**

```
[入口] カード選択（場所・時間帯）1〜2問
         ↓
[対話] AIチャットが気分・目的を深掘り
         ↓
[推薦] セマンティック検索で商品マッチング + 理由文表示
```

### 核心的な差別化UX: **リアルタイム香りプロファイル表示**

チャット画面の横に、AIが理解した内容をリアルタイムで整理して表示する。

```
┌──────────────────────┬──────────────────────────────┐
│  💬 AIチャット         │  📋 あなたの香りプロファイル   │
│                      │                              │
│ AI: 夜に眠れないんですね│  📍 場所   : 寝室            │
│ 考えすぎてしまう感じ？ │  ⏰ 時間帯 : 夜・就寝前       │
│                      │  😔 状態   : 考えすぎ・緊張   │
│ User: そうなんです…   │  🎯 目的   : 思考を止めたい   │
│                      │                              │
│                      │  💡 AIより                   │
│                      │  「リラックス」よりも           │
│                      │  「思考リセット」向けが合いそう │
└──────────────────────┴──────────────────────────────┘
```

**この体験が生むもの：**
- 「わかってもらえてる」感 → 信頼感
- 自分の状態を客観視できる → 自己理解の促進
- AI提案の根拠が見える → 受け入れやすさ

## 技術スタック（暫定）

- **フロントエンド**: Streamlit（爆速プロトタイプ）or Next.js
- **AIバックエンド**: Claude API（対話 + プロファイル抽出 + 推薦）
- **商品検索**: 商品JSONをembedding化 → セマンティックサーチ
- **プロファイル生成**: 会話ログをLLMで構造化（場所・時間・目的・強さ）

---

## 確定した設計方針

### 6軸フレームワーク（内部ロジック）
対話中はユーザーに見せず、AIが会話から自動的に更新する。

| 軸 | 両端 |
|---|---|
| 生理的 | 活性（ON） ↔ 鎮静（OFF） |
| 感覚的 | クール・爽やか ↔ ウォーム・甘め |
| 社会的 | パブリック・コミュニケーション ↔ プライベート・集中 |
| ブランド | フォーマル・信頼 ↔ カジュアル・革新 |

※KAORILOGOへの訪問後、軸の内容は修正される可能性あり

### レーダーチャートの使い方
- 対話中：ユーザーには非表示（AIが内部で軸スコアを更新）
- 推薦結果画面で初めて表示
- 「なぜこの香りか」の納得感・購入後押しとして機能

```
推薦結果イメージ:
  🌿 あなたにはこの香りが合いそうです

       集中 ●────────●
      爽やか ●───●
     リラックス ●──●

  「頭がシャープになるような爽やかでクリアな香りです」
  [購入ページへ]
```
